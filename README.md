# Personal Website of Himanshu Sandha

Built using React

# Get started

<ul>
  <li>Clone this repo</li>
</ul>
   
    git clone git@github.com:himasnhusandha/himanshusandha.com.git

    cd himanshusandha.com

    yarn install

    yarn start
 
<ul>
  <li>Visit <code>http://localhost:3000</code></li>
</ul>
